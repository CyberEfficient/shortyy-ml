A Brand URL Shortener

shortyy.ml is a url shortener that allows you to shorten your links and giving them a brand url. Its completely free. But if your links are found to be violating our terms and services we may remove the links from our database. This is started as a project. The source code of this will be available on our github repository. This is made in Python using flask as web framework and postgres as database and tailwind as CSS framework.
